- Resolution: High (native device resolution)

After capturing, you can add arrows/labels pointing to:
- "Fiuu QR Code appears here"

## Quick Capture Commands (Windows)

### Screenshot Shortcuts:
- **Windows Key + Shift + S**: Snipping tool (select area)
- **Windows Key + Print Screen**: Full screen screenshot
- Screenshots saved to: `Pictures\Screenshots\`

### Or Use Android Studio Emulator:
- Click camera icon in emulator toolbar
- Screenshots saved to: Project folder or Desktop

## Troubleshooting

### App Won't Start?
```powershell
cd portable-refill-app
rm -rf node_modules
npm install
npm start
```

### Wallet Top-Up Screen Doesn't Exist Yet?
Check if `top-up-wallet.tsx` is implemented:
```powershell
ls app/(tabs)/top-up-wallet.tsx
```

If it exists, the screen is built. If not, you may need to create mockups instead.

### Can't Login?
Use test credentials from code:
```typescript
// From login.tsx
email: 'tester@acerev.my'
password: 'AceRev@2026'
```
