## 🚀 3 SIMPLE STEPS TO PUBLISH

### STEP 1: Upload Policy Pages (5 minutes)

1. Go to: https://app.netlify.com/drop
2. Drag BOTH files together:
   - `privacy-policy.html`
   - `delete-account.html`
   - `corporate-kiosk-policies.html` (optional, recommended)
3. Netlify gives you a URL like: `https://acerev-12345.netlify.app`
4. **Save these URLs:**
   - Privacy: `https://YOUR-SITE.netlify.app/privacy-policy.html`
   - Delete: `https://YOUR-SITE.netlify.app/delete-account.html`
   - Terms: `https://YOUR-SITE.netlify.app/corporate-kiosk-policies.html`

---

### STEP 2: Generate Feature Graphic (2 minutes)

**Chrome is already open with the feature graphic!**

Just follow these steps:
1. Press **F12** (opens DevTools)
2. Press **Ctrl+Shift+P** (opens command menu)
3. Type: `screenshot`
4. Click: **"Capture full size screenshot"**
5. Save as: `feature-graphic.png` in this playstore folder

**Done! You now have your 1024x500 feature graphic.**
